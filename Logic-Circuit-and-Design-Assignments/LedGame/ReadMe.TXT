

   For the led game,the requirements are as follows:

    1. In this game, the machine has 7 LEDs and two buttons for two players. 
   
    2. The game starts with the led in the middle turned on. 

    3. One user tries to shift the ON led to left while the other tries to push it to right. 

    4. Winning occurs when the turned on led is at the rightmost or at the leftmost position.

    5. The machine shows the score as Player1 vs Player2.

    5. Also, machine has a reset input to start a new game.
